## Debugging Go in VS Code

- Prerequisites: Install [Go](https://go.dev/doc/install) and the VS Code [Go extension](https://marketplace.visualstudio.com/items?itemName=golang.Go).

- Quick start: open the project, set a breakpoint, then press F5 to start debugging.

- Notes: the extension uses Delve (`dlv`). For `launch.json` attributes and examples, see [config](https://github.com/golang/vscode-go/wiki/debugging#launchjson-attributes)
